KODI更换地平线皮肤

转帖链接

1. 引子

之前介绍过v1版本的Arctic.Horizon受到了很多吧友的好评，原作者22年3月又出新作，皮肤延续了上代简约时尚的扁平风格，相对于v1版本更加强调了影片内容的展示，优化了交互菜单的质感，更重要的是，此次加入了针对中文、韩文、日文的字符集。在切换字符集后很好的利用了官方中文汉化库，虽然还有些内容未汉化，这个我会在较晚时间推出汉化后的版本，但水平有限，仅供参考。

2. 皮肤介绍

皮肤名称 ARCTIC HORIZON 2

作者git主页 /jurialmunkey
ARCTIC HORIZON 2地址

3. 皮肤预览



4. 安装kodi 熟悉的朋友可以忽略

4.1 kodi官网选择自身的平台进行下载 版本号目前我测试的平台win和mac M1,kodiv19、v20 都可以正常使用。
4.2 安装完毕示意图 即默认的kodi状态

5. 安装皮肤前置说明

5.1 我此处安装以全新安装kodi后未进行任何操作的前提进行安装，目的是为了排除其他插件版本引起的各种问题，可以保证能够安装成功，若在安装过程中出现了其他未列举出的情况，欢迎留言讨论
5.2 安装方法
5.2.1 通过原作者提供的库进行安装，此方法安装非常简单，但需要网络通畅，各位可以自行了解。
5.2.2 通过我帖子中提供的完整依赖压缩包安装，步骤相对于使用库安装较为复杂，但按照步骤来，是可以安装成功的，需要仔细的按照教程一步步操作

6. 通过压缩包安装

6.1 资源链接
a. 链接: https://pan.baidu.com/s/1DbRyXUc3zNP84KPos-dPSQ?pwd=s8dq 提取码: s8dq 复制这段内容后打开百度网盘手机App，操作更方便哦
6.2 资源介绍

由于使用安装包安装相当于手动安装全部的依赖，对安装顺序有一定的要求，因此我将压缩包名称最前面使用数字表示安装顺序，如 1.script.module.tmdbhelper-0.0.7.zip 即第一个安装的包
6.3 开始安装 下载压缩包后解压到自己方便找到的位置即可开始安装
6.4 打开kodi -> 点击 设置（齿轮图标）-> Add-ons (安装插件) -> Install from zip file (从zip文件安装)

6.5 此时会弹窗询问你安装压缩包需要同意安装第三方插件 点击 seeting 跳转至开关 打开开关 并同意

6.6 返回上一页面 继续选择zip压缩包安装 找到解压后的安装包 点击则开始安装 有的包安装时间会比较久 注意路径不能有中文 不然会显示空格

6.7 插件安装成功 屏幕右上方 提示Add-on installed 则安装成功 若提示安装失败或其他 请留言讨论

6.8 按照安装包的顺序安装完毕 最后一个就是皮肤的压缩包 安装过程可能会比较长 需要耐心等待
6.9 皮肤安装成功 会提示是否切换到此皮肤 点击切换

6.10 切换后询问是否保持使用此皮肤 点击同意

6.11 经过一小会加载后 皮肤安装完成了 这就是安装完毕的初始样式 扑面而来的现代感，比默认的皮肤不知道好看多少倍了。

7. 皮肤设置

7.1 关于中文设置
为什么没有在安装一开始就设置中文呢？因为在默认的皮肤中，设置中文后直接切换皮肤会导致字符集错误，表现为全是空格，所以我们在切换皮肤后再来修改语言。在这个v2版本中，加入了对中文字符集的支持，使得皮肤可以使用官方的一部分汉化内容
7.2 设置中文字符集 点击右上角的齿轮图标 -> seeting -> 选择 skin-> Fonts 修改字符集 点击后会卡顿一下 等待一会儿即可

7.3 设置中文 点击Regional -> Language -> 选择 Chinese-(SimSimple) 简体中文 等待安装即可

8. 皮肤安装完成 一些官方自带的文案已经替换成中文了 在设置中和皮肤相关性比较大的会表现为英文


至此安装就完成了 至于皮肤的设置就由各位自由发挥了
关于使用库的安装方法，可以自行了解，其实按照这个压缩包安装已经非常简单了，我是查看源码一点点找出所有依赖才安装成功的。
